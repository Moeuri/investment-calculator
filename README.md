## 功能說明

- 📈 **定期定額**：009816 vs 0050，含一次性投入、配息再投入效率拉桿、六檔歷史錨點報酬率
- ⚖️ **0050 vs 009816**：選股邏輯、持股限制、配息機制、費用率、風險分析完整比較
- 💥 **崩盤模擬**：三次崩盤疊加，五大歷史事件預設，V/U型恢復模型
- 💰 **通膨購買力**：名目 vs 實質資產，儲蓄險年領購買力侵蝕試算
- 🏖️ **退休提領**：退休時間設定、衝突偵測、退休本金反推（年金現值）
- 🎯 **高股息ETF**：0056 / 00878 / 00919 月曆式配息、目標月領反推
- 🏦 **儲蓄險 vs 股市**：保險本金解約轉股市的長期比較

---

## 免責聲明

本計算器僅供個人財務規劃參考，不構成任何投資建議。  
所有報酬率均基於歷史數據估算，過去績效不代表未來結果。  
投資人應自行評估風險承受能力。
